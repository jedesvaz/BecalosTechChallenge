
# Proyecto Final de Introducción a la Programación  
### Bécalos Fundación Traxión Tech Challenge


## Propósito del Proyecto
Este proyecto consiste en la creación y publicación de una página web básica utilizando HTML y CSS, desplegada a través de GitHub Pages.  
Su objetivo es demostrar el entendimiento en la construcción de páginas web, la organización de elementos esenciales del código y la práctica en el uso de Git y GitHub para el control de versiones y la publicación de proyectos.
Se utiliza contacto de tienda de mascotas como aplicación de lo aprendido en el curso y como forma de adelantar el proyecto de landing page para tal negocio



## Objetivo
El proyecto busca reforzar los conocimientos adquiridos en el bloque Introducción a la Programación del programa Bécalos Fundación Traxión Tech Challenge, aplicando habilidades prácticas en el desarrollo web.


## Tecnologías Utilizadas
- HTML5  
- CSS3

## Requisitos Previos
No se requieren dependencias ni instalaciones adicionales.  
El proyecto puede visualizarse directamente desde el navegador.
